## 1. Executive Summary
- ДҗГўy lГ  hб»Ү thб»‘ng nhбәӯn diб»Үn kГҪ tб»ұ (OCR) Д‘Ж°б»Јc xГўy dб»ұng Д‘б»ғ nhбәӯn diб»Үn vГ  convert thГҙng tin tб»« file upload cб»§a ngЖ°б»қi dГ№ng 
- Hб»Ү thб»‘ng Д‘Ж°б»Јc xГўy dб»ұng dб»ұa trГӘn cГЎc nguyГӘn tбәҜc thiбәҝt kбәҝ tбәӯp trung vГ o viб»Үc bбәЈo mбәӯt dб»Ҝ liб»Үu vГ  serverless. TrГЎnh viб»Үc cГЎc module lЖ°u lбәЎi thГҙng tin 
- Triб»ғn khai theo kiбәҝn trГәc orchestration nЖЎi cГі mб»ҷt module chuyГӘn Д‘б»ғ Д‘iб»Ғu phб»‘i luб»“ng dб»Ҝ liб»Үu giб»Ҝa cГЎc service
- Triб»ғn khai sб»ӯ dб»Ҙng hбәЎ tбә§ng cloud
### Design Principles
| Principle | Description |
|-----------|-------------|
| **Serverless** | GiбәЈm thiбәҝu khбәЈ nДғng cГЎc service biбәҝt Д‘Ж°б»Јc cГҙng viб»Үc cб»§a nhau |
| **Save cost** | Tб»‘i Ж°u chi phГӯ khi triб»ғn khai hбәЎ tбә§ng cloud |
| **Security** | BбәЈo mбәӯt dб»Ҝ liб»Үu vГ  trГЎnh rб»§i ro bб»Ӣ tбәҘn cГҙng, Д‘ГЎnh cбәҜp thГҙng tin |
| **Scalable** | Hб»Ү thб»‘ng cГі thб»ғ mб»ҹ rб»ҷng khi cбә§n |
