## time: 16:58

### Mục tiêu
Debug và fix MK-011 failed test (Vietnamese filename → 500 error)

### Kết quả

**Root cause:** `Content-Disposition` header trong `/jobs/{id}/download` endpoint chứa Vietnamese filename (`thiệp mừng năm mới_result.txt`). HTTP headers chỉ support Latin-1 encoding → `UnicodeEncodeError: 'latin-1' codec can't encode character '\u1ec7'`.

**Fix:** RFC 6266 compliant `Content-Disposition` — ASCII fallback `filename` + UTF-8 encoded `filename*`:
```
Content-Disposition: attachment; filename="result_8d733f58.txt"; filename*=UTF-8''thi%E1%BB%87p%20m%E1%BB%ABng%20...
```

**Kết quả: 12/12 E2E tests PASSED**
