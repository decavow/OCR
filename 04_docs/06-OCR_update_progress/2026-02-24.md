### TASK 1: CORS — Cấu hình whitelist origins
- **Mức độ:** CRITICAL
- **File:** `02_backend/app/main.py` (dòng ~30)
- **Hiện tại:** `allow_origins=["*"]` — chấp nhận mọi origin
- **Yêu cầu:** Whitelist origins, đọc từ env var
- **Hướng sửa:**
  - Đọc `CORS_ORIGINS` từ environment variable (comma-separated)
  - Default cho dev: `http://localhost:5173` (Vite dev server)
  - Giữ `allow_credentials=True`, `allow_methods=["*"]`, `allow_headers=["*"]`
- **Effort:** ~15 phút

### TASK 2: Batch size — Validate tổng dung lượng 200MB
- **Mức độ:** HIGH
- **File:** `02_backend/app/modules/upload/validators.py` (dòng ~47)
- **Hiện tại:** Check 50MB/file + max 20 files. Không check tổng → có thể upload 1GB (20 × 50MB)
- **Yêu cầu:** Tổng input size không vượt 200MB
- **Hướng sửa:**
  - Thêm constant `MAX_TOTAL_BATCH_SIZE = 200 * 1024 * 1024`
  - Trong `validate_batch()`, tính `total_size = sum(len(content) for content in files)`
  - Raise lỗi nếu `total_size > MAX_TOTAL_BATCH_SIZE`
  - Cập nhật frontend config `MAX_TOTAL_SIZE` để hiển thị cảnh báo trước khi upload
- **Effort:** ~30 phút

- **Effort:** ~1 giờ
